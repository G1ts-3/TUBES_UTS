## ğŸš€ Cara Menjalankan

Berikut adalah langkah-langkah untuk meng-compile dan menjalankan program ini:

1.  Clone repositori ini:
    ```bash
    git clone [https://github.com/G1ts-3/TUBES_UTS.git](https://github.com/G1ts-3/TUBES_UTS.git)
    ```
2.  Masuk ke direktori proyek:
    ```bash
    cd TUBES_UTS
    ```
3.  Kompilasi (Compile) semua file `.java` di dalam folder. Perintah ini akan membuat file `.class` untuk setiap file `.java`:
    ```bash
    javac *.java
    ```
    *Catatan: Jika ada error, pastikan Anda sudah menginstall Java Development Kit (JDK) di komputer Anda.*

4.  Jalankan program. Ganti `[NamaFileUtama]` dengan nama file Anda yang berisi method `public static void main(String[] args)`.
    ```bash
    java [NamaFileUtama]
    ```
    **Contoh:**
    Jika file utama Anda adalah `SistemAntrianKlinik.java` (dan di dalamnya ada `main`), maka perintahnya adalah:
    ```bash
    java SistemAntrianKlinik
    ```
    *(Perhatikan: Perintah `java` tidak menggunakan ekstensi `.java` atau `.class` di akhir nama filenya).*
